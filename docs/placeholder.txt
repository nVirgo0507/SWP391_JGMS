3.1 Team Leader Processes
A. Manage Group Requirements
Create requirement in Jira → Input details → Set priority → Create via API
Edit requirement → Update fields → Sync with Jira
Delete requirement → Confirm → Remove from Jira
Organize requirements → Arrange hierarchy (Epic → Story → Task)
B. Manage Tasks
Create task → Assign to member → Set deadline → Add to sprint
Edit task → Update status, assignee, or description → Sync changes
Move tasks between sprints → Drag and drop → Update via API
Link tasks to requirements → Create relationships in Jira
C. Monitor Task Progress
View team dashboard → Display all tasks with status indicators
Track sprint progress → Show burndown chart → Highlight blockers
View member workload → Display tasks per member → Check balance
Identify bottlenecks → Analyze overdue tasks → Generate alerts
3.2 Jira Integration Implementation
A. Jira API Integration
Authenticate with Jira Cloud → Handle OAuth/API tokens
Fetch projects, issues, sprints → Map to internal data structure
Create/Update/Delete operations → CRUD via REST API
Webhook handling → Listen for Jira events → Update local database
Sync mechanism → Schedule periodic sync → Handle conflicts
Error handling → Retry logic → Log failures
B. Backlog Management
Fetch backlog items → Display prioritized list
Sprint planning → Move items to sprint → Update via API
Track sprint capacity → Calculate story points → Warn on overload

Team Member Functions & GitHub Integration
4.1 Team Member Processes
A. View Assigned Tasks
Login → Load personal dashboard → Display assigned tasks
Filter tasks → By status, priority, or sprint → Update view
View task details → Display full description, comments, attachments
Sort tasks → By due date, priority → Reorder display
B. Update Task Status
Select task → Change status (To Do → In Progress → Done)
Add comments → Input text → Post to Jira via API
Log work hours → Input time spent → Update Jira work log
Attach files → Upload → Link to task in Jira
C. View Personal Statistics
Task completion dashboard → Show completed vs. pending
Personal commit history → Fetch from GitHub → Display timeline
Contribution metrics → Lines of code, commit frequency
Performance trends → Weekly/monthly comparison charts
